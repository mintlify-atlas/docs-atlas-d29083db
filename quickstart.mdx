## Key Simulation Parameters

The simulation framework is controlled by several core parameters defined in the main execution block:

### Core Parameters

```python
theta_param = 0.70        # Pivotality threshold (70%)
R_param = 100             # Number of replications per scenario
max_preds_param = 12      # Maximum predecessors for P2 calculation
seed_param = 42           # Random seed for reproducibility
```

- **theta** (`0.70`): The pivotality threshold. A node is pivotal if its weight exceeds `(1 - theta) * T_j` where `T_j` is the total inflow to target `j`
- **R** (`100`): Number of Monte Carlo replications for each scenario
- **max_preds** (`12`): Cap on non-pivotal predecessors for tractable P2 (pairwise pivotality) computation
- **seed** (`42`): Random seed ensuring reproducible results

### Network Scenarios

The simulation runs across multiple bipartite network configurations with varying exporter (M) and importer (N) counts:

```python
scenarios = [
    (140, 60),   # Exporter-heavy
    (130, 70),
    (120, 80),
    (110, 90),
    (100, 100),  # Balanced
    (90, 110),
    (80, 120),
    (70, 130),
    (60, 140),   # Importer-heavy
]
```

The edge probability `p` is dynamically adjusted to maintain a constant expected target in-degree:

```python
target_degree = 20
def p_rule(M, N):
    p = target_degree / M
    return float(min(max(p, 0.0), 1.0))
```

### Weight Distributions

The framework tests three different weight distributions:

1. **Gaussian** (`"gaussian"`):
   - Truncated normal distribution
   - Parameters: `μ = 100`, `σ = 20`, truncated at 0
   - Light-tailed, homogeneous weights

2. **Student's t** (`"student_t"`):
   - Student's t-distribution with fat tails
   - Parameters: `df = 3`, `loc = 100`, `scale = 20`
   - Medium-tailed distribution

3. **Power Law** (`"power_law"`):
   - Pareto distribution
   - Parameters: `shape = 2`, `scale = 50`
   - Heavy-tailed, highly heterogeneous weights

### Understanding the Metrics

The simulation computes:

- **P1**: Count of targets for which an exporter is individually pivotal
- **P2**: Count of pairwise pivotal relationships (capped for tractability)
- **Gini coefficient**: Concentration of pivotality (0 = equal, 1 = concentrated)
- **Top 5% share**: Fraction of total pivotality held by top 5% of exporters
- **frac_P1_zero**: Proportion of exporters with no individual pivotality

<Note>
  Under Gaussian weights, pivotal nodes are extremely rare. Under power-law weights, pivotality becomes heavily concentrated in a few nodes, demonstrating the emergence of critical nodes in heavy-tailed networks.
</Note>
