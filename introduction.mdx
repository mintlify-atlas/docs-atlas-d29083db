# Bipartite Tail Pivotality

This repository contains the simulation framework used in:

**Quota-Based Centrality under Heavy-Tailed Weight Regimes**  
*Brian Daniel Bernhardt (2026)*

<Note>
This framework provides a complete toolkit for studying pivotality phenomena in bipartite networks with different weight distributions.
</Note>

## Overview

The framework evaluates pivotality measures in random bipartite networks under three distinct weight regimes:
- **Truncated Gaussian** (light-tailed)
- **Student-t** with df=3 (moderate heavy-tail)
- **Pareto** power-law (extreme heavy-tail)

## What You Can Simulate

The framework computes comprehensive graph-level statistics:

- **max P1** — Maximum single-node pivotality
- **frac(P1 = 0)** — Fraction of nodes with zero pivotality
- **Gini(P1)** — Inequality measure for P1 distribution
- **max P2** — Maximum pair pivotality
- **frac(P2 = 0)** — Fraction of nodes with zero pair pivotality
- **Conditional concentration metrics** — Analysis restricted to pivotal nodes

<Info>
All simulations use fully synthetic data. No empirical datasets are required.
</Info>

## Research Context

This simulation framework supports the analysis of:

1. **Single-node pivotality (P1)** — When does removing one exporter cause an importer to fall below their quota threshold?
2. **Pair pivotality (P2)** — How many importer pairs does an exporter simultaneously pivot?
3. **Exposure amplification** — How do heavy-tailed weights amplify concentration as bipartite ratios change?
4. **Structural transitions** — Visualizing pivotality evolution across (M, N) scenarios
